# Srijan Digital Engine 2026

A high-fidelity, production-ready portfolio for **Srijan Ghosh** (Data Engineer & AI Specialist), built using Astro 5 and Tailwind CSS v4.

## Design Aesthetic: Tech-Minimalist
- **Themes**: Deep Midnight (Dark Mode) & High-Contrast (Light Mode).
- **Color Palette**: Cyber Citrus (`#ccff00`) accents on Data Slate (`#1A1A1A`) and Dark (`#0D0D0D`) backgrounds.
- **Typography**: `Inter` for clean body text and `JetBrains Mono` for headers/technical metrics.

## Tech Stack
- [Astro](https://astro.build/) - Web framework
- [Tailwind CSS v4](https://tailwindcss.com/) - Styling
- [React](https://react.dev/) - Interactive components (installed, ready for future use)

## Getting Started
To run the development server locally:

```bash
npm install
npm run dev
```

Then visit `http://localhost:4321` in your browser.
